Some Important Concepts to know -
1. substring - A substring kisi string ka continuous (contiguous) part hota hai.
Matlab characters ek saath hone chahiye, skip nahi kar sakte.
String: "hello"
Valid Substrings: "", h", "he", "e"
kitne substring ho sakte hai - (n*(n+1)/2)+1 (where n is the size of the string)
Maintain order - yes
Allowed skips - No skips
Generated by - Sliding window

2. subsequences - A subsequence kisi string ya array ka ordered subset hota hai, jisme
characters/values continuous hone ki zaroorat nahi hoti. Lekin, order maintain rehna chahiye
(i.e., characters ke relative positions same hone chahiye).
String - "abc"
Valid subsequences: "", "ac", "bc"
kitne subsequences ho sakte hai - 2^n (where n is the size of the string)
Maintain order - yes
Allowed skips - Yes
Generated by - Recursion / DP

3. subsets - Agar order follow nahi karna, toh usko "Subset" kehte hain.
Matlab elements ko kisi bhi order me le sakte ho, bas duplicates nahi hone chahiye.
String -  "abc"
Valid subsets: "", {'a', 'b'}, {'c', 'a'}, {'b', 'a'}
kitne subsets ho sakte hai - 2^n (where n is the size of the string)
Maintain order - no
Allowed skips - Yes
Generated by - Pick/NotPick (known as Backtracking)
